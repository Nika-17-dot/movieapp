Movie App - Android Final Project

ეს არის ფილმების აპლიკაცია, რომელიც იყენებს **TMDB API**-ს.

ფუნქციები:
* პოპულარული ფილმების სია.
* ფილმების ძებნა დასახელების მიხედვით.
* ფავორიტების სია (შენახვა ლოკალურ ბაზაში - Room DB).
* ფავორიტი ფილმის წაშლა დიდხანს დაჭერით (Long Click).

ტექნოლოგიები:
* Retrofit & Gson**: მონაცემების წამოსაღებად.
* Glide**: სურათების ჩასატვირთად.
* Room Database**: ლოკალური შენახვისთვის.
* Coroutines**: ასინქრონული ოპერაციებისთვის.
* View Binding**: UI ელემენტებთან სამუშაოდ.

**შენიშვნა:** აპლიკაციის გასაშვებად საჭიროა TMDB API Key.
